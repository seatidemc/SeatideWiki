# 下载内容

以下是目前对外开放的下载内容，点击链接即可下载。

# modpack.7z (opens new window)

内含本周目所需的所有模组。解压到 mods 文件夹即可进入服务器。

# client-simple.7z (opens new window)

简单整合包，适用于新手快速开始游戏。内含 modpack 的所有内容+简单的辅助模组+两个启动器（PCL 和 HMCL，根据个人喜好选择）。

# ServerIpUpdate-1.0.jar (opens new window)

由 SEATiDE 玩家所作。一个用于自动获取服务器 IP 的模组，安装后自动刷新 IP。这样就不需要每次前往官网的状态页面 (opens new window)查看 IP 了。
